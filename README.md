# Netflix Data Analysis

This is a beginner data analysis project using a Netflix dataset. The project was made inside the DataCamp website as part of my learning and practice in data analysis.

## What I Did

In this project, I explored the Netflix dataset to better understand the data and find simple patterns. I used Python to load the dataset, check the columns, clean the data, and create basic visualizations.

## Tools Used

- DataCamp
- Python
- pandas
- NumPy
- matplotlib
- Jupyter Notebook

## Files in This Project

- `notebook.ipynb` - contains the Python code and analysis
- `netflix_data.csv` - contains the dataset used for the project
- `README.md` - explains the project

## Skills Practiced

- Loading a CSV file using pandas
- Viewing and understanding a dataset
- Checking missing values
- Filtering data
- Using basic pandas functions
- Creating simple charts with matplotlib
- Writing observations from the data

## What I Learned

This project helped me practice working with real data through DataCamp. I learned how to explore a dataset, use pandas to organize information, and create charts to make the data easier to understand.

## How to Run the Project

1. Download the project files.
2. Open `notebook.ipynb` using Jupyter Notebook, VS Code, Google Colab, or DataCamp.
3. Make sure `netflix_data.csv` is in the same folder as the notebook.
4. Run the notebook cells from top to bottom.
